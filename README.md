## CalculatorProject
This is calculator project built with html and css. This project perform basic arithmetic like Addition, subtraction, multiplication, and division. This project is very friendly for beginner

---
## Technologies used
* html
* Css
* Javascript
---

## Setup and Installations
1. Clone the repository:

```bash
git clone https://github.com/your-username/calculator-project.git
```

2. Navigate to the project directory:

```bash
cd calculator-project
```

3. Open `index.html` in your browser.
